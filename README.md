# Directorio de Marcas - Proyecto POO

Este proyecto es el desarrollo práctico de un Directorio de Marcas utilizando conceptos de Programación Orientada a Objetos (POO) en Java y una interfaz de presentación estática en HTML/CSS.

## Estructura del Proyecto

```text
directorio-marcas/
│
├── index.html          # Página principal del directorio
├── css/
│   └── styles.css      # Hoja de estilos de la página
│
├── java/
│   ├── Marca.java             # Clase modelo con encapsulamiento
│   ├── DirectorioMarcas.java  # Clase gestora con Listas
│   └── Main.java              # Clase de ejecución principal
│
└── README.md           # Documentación del proyecto
